# 반응형 Navbar

HTML+CSS 실습 목적으로 만든 반응형 navbar 👉
[Live Demo](https://hyunji-lee-dev.github.io/navbar/)

## 사용기술

- HTML
- CSS
- JavaScript

## 구현사항

- 반응형 디자인: flexbox, 반응형 단위, media query
